Reading Data from a File

Real clinical datasets live in files, not Python objects. pandas can read CSV, Excel, JSON, and many other formats. In an ideal world every file would be perfectly formatted — but real-world health data exports rarely are. This section walks through reading a messy file step by step, fixing one issue at a time.

nhs_readmission.csv dataset

This dataset is entirely synthetic, generated purely as a teaching example.

  • Simulated distributions based on typical NHS inpatient population parameters (e.g. age ~Normal(62, 16), length of stay ~Exponential(4.5), HbA1c ~Normal(60, 18))
  • Clinically plausible missingness patterns (HbA1c missing in ~16% of patients, eGFR in ~9%)
  • A logistic outcome model where readmission probability depends on age, Charlson index, length of stay, prior ED attendances, eGFR, and IMD decile
Column descriptions
Column Description
patient_id Anonymous patient identifier
age Age in years at admission
sex M / F
imd_decile Index of Multiple Deprivation (1 = most deprived, 10 = least)
primary_diagnosis ICD-10 code for primary diagnosis
charlson_index Charlson Comorbidity Index (0–37)
hba1c Most recent HbA1c (mmol/mol) — missing if not tested
egfr Most recent eGFR (mL/min/1.73m²) — missing if not tested
bmi Body Mass Index (kg/m²)
los_days Length of stay in days
ed_prior_year ED attendances in previous 12 months
discharge_ward Ward the patient was discharged from
readmitted_30d Readmitted within 30 days: Yes / No

Step 2 — Skip the header rows

We use skiprows=3 to ignore the first 3 lines before looking for column names:

df = pd.read_csv(
    url,
    skiprows=3,  # Add this
)

Step 3 — Set the row index

By default pandas assigns integer row numbers (0, 1, 2, …). Since patient_id uniquely identifies each row, it makes a better index:

df = pd.read_csv(
    url,
    skiprows=3,
    index_col="patient_id",  # Add this
)

Step 4 — Handle missing values

We noticed that the file uses ., N/A, NULL, and UNKNOWN as missing markers — pandas won’t recognise these automatically. We fix this with na_values:

df = pd.read_csv(
    url,
    skiprows=3,
    index_col="patient_id",
    na_values=[".", "N/A", "NULL", "UNKNOWN"],  # Add this
)

print("Missing values per column:")
print(df.isnull().sum())

Why is data missing?

Missing hba1c (≈16%) likely means the test was not ordered — common in surgical or trauma admissions. Missing egfr (≈9%) may mean no recent blood tests. In machine learning, “missing” can itself be an informative signal. Always think about why data is absent before deciding how to handle it.

groupby() — Aggregation by group

groupby() splits the dataset into groups and applies a function to each — the pandas equivalent of a PivotTable.

df.groupby("readmitted_30d")[["age", "charlson_index", "los_days"]].agg(["mean", "median"]).round(1)
