Introduction to Pandas

⏱ 15 minutes

Pandas is a core Python library for working with structured (tabular) data. It provides high-level data structures like DataFrame and Series, designed to make data cleaning, transformation, and analysis efficient and expressive.

With pandas, you can filter rows, compute aggregates, reshape datasets, merge multiple sources, and handle missing values using concise, readable code. Because every step of your workflow is written as code, analyses can be rerun, reviewed, version-controlled, and shared without ambiguity. It also scales well to large datasets and integrates seamlessly with visualization libraries and machine learning frameworks.

The DataFrame

The core pandas object (data structure) is the DataFrame — a two-dimensional table with labelled rows and columns.

import pandas as pd

# A minimal patient DataFrame built from a dictionary
data = {
    "patient_id":  ["P001", "P002", "P003", "P004"],
    "age":         [67, 45, 82, 55],
    "sex":         ["M", "F", "F", "M"],
    "diagnosis":   ["E11", "I10", "J44", "E11"],  # ICD-10 codes
    "hba1c":       [71, None, 58, 84],            # mmol/mol; None = missing
    "readmitted":  [True, False, True, False],
}

df = pd.DataFrame(data)
print(df)

Accessing Columns

Select a single column using square brackets — this returns a Series:

ages = df["age"]
print(ages)
print(type(ages))
0    67
1    45
2    82
3    55
Name: age, dtype: int64
<class 'pandas.core.series.Series'>

Select multiple columns by passing a list of names — this returns a DataFrame:

demographics = df[["patient_id", "age", "sex"]]
print(demographics)
  patient_id  age sex
0       P001   67   M
1       P002   45   F
2       P003   82   F
3       P004   55   M

Creating Columns

Assign to a new column name to add it to the DataFrame. The value can be a constant, a calculation on existing columns, or a condition:

df["age_over_65"] = df["age"] > 65
df["hba1c_above_target"] = df["hba1c"] - 48  # target threshold: 48 mmol/mol
print(df[["patient_id", "age", "age_over_65", "hba1c", "hba1c_above_target"]])
  patient_id  age  age_over_65  hba1c  hba1c_above_target
0       P001   67         True   71.0                23.0
1       P002   45        False    NaN                 NaN
2       P003   82         True   58.0                10.0
3       P004   55        False   84.0                36.0

Series Operations

A Series supports vectorised operations — you apply a function to every element at once. They use optimised C code under the hood, making them much faster than Python for loops. For large datasets with hundreds of thousands of rows, this matters.

# Classify each patient's age in a single line
age_over_65 = df["age"] > 65
print(age_over_65)
0     True
1    False
2     True
3    False
Name: age, dtype: bool
0     True
1    False
2     True
3    False
Name: age, dtype: bool

Common Series methods:

df["age"].mean()      # average age
np.float64(62.25)
df["age"].median()    # median age
61.0
df["age"].std()       # standard deviation
15.945218719101975
df["age"].min()       # minimum
45
df["age"].max()       # maximum
82
df["hba1c"].count()   # non-missing count
np.int64(3)
df["sex"].value_counts()  # frequency of each category
sex
M    2
F    2
Name: count, dtype: int64
